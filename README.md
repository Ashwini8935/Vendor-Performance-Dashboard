# Vendor-Performance-Dashboard
Vendor Performance Dashboard using Excel and Power BI
## Project Overview
Built a procurement analytics dashboard to evaluate vendors based on delivery performance, lead time, defects, and purchase value.

## Tools Used
- Excel / CSV
- Power BI

## Key Features
- Purchase Value by Vendor
- Delivery Status Analysis
- Average Lead Time
- Defect Analysis
- Monthly Procurement Trend

## Business Impact
Helps improve supplier selection, reduce delays, and control procurement costs.
